# SQL Exploratory Data Analysis Project

## Introduction

After cleaning the layoffs dataset in Project 1, this project focuses on exploring the data to uncover meaningful business insights. Using SQL, the analysis investigates global layoff trends across companies, industries, countries, funding stages, and time to better understand how layoffs evolved throughout the dataset.

This project demonstrates how SQL can be used not only to retrieve data, but also to perform powerful analytical queries that support data-driven decision making.

## Skills Showcased

* **📊 Exploratory Data Analysis (EDA):** Analyzing real-world business data to identify trends and patterns.
* **📈 Aggregate Functions:** Using `SUM()`, `MAX()`, `MIN()`, and `COUNT()` to summarize data.
* **🪟 Window Functions:** Applying rolling totals and advanced analytical calculations.
* **🏆 Ranking Functions:** Using `DENSE_RANK()` to identify top-performing companies and countries by year.
* **⚙️ Common Table Expressions (CTEs):** Structuring complex analytical queries for readability and maintainability.
* **📅 Time-Series Analysis:** Examining layoffs across months and years.
* **📉 Rolling Calculations:** Calculating cumulative layoffs over time using Window Functions.
* **💼 Business Insight Generation:** Identifying trends across companies, industries, countries, and funding stages.

## Analysis Overview

### Company Analysis

Explore which companies experienced the largest layoffs and compare yearly layoff totals using aggregate functions and ranking techniques.

### Industry Analysis

Identify industries most affected by layoffs and compare overall workforce reductions across sectors.

### Country Analysis

Analyze layoffs geographically to understand which countries experienced the greatest workforce reductions.

### Time-Series Trends

Track layoffs over time using monthly and yearly aggregations, including rolling cumulative totals to visualize overall trends.

### Ranking Analysis

Window Functions and `DENSE_RANK()` are used to determine the top companies and countries with the highest layoffs for each year.

## Conclusion

This project demonstrates how SQL can transform a cleaned dataset into meaningful business intelligence through exploratory data analysis. By combining aggregations, Common Table Expressions, Window Functions, and ranking techniques, the project uncovers valuable trends that support data-driven decision making and showcases practical SQL skills commonly used in data analytics roles.
